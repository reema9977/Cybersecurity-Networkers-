# 📘 Markdown Ultimate Guide

This is your reference for writing professional notes on GitHub.

---

## 1. Headings (العناوين)
# Heading 1
## Heading 2
### Heading 3

---

## 2. Text Styling (تنسيق النص)
* **Bold:** `**Text**`
* *Italic:* `*Text*`
* ~~Strikethrough:~~ `~~Text~~`

---

## 3. Lists (القوائم)
* Bullet point 1
* Bullet point 2
  * Sub-point (Press Tab)

1. Numbered item 1
2. Numbered item 2

---

## 4. Code (الأكواد)
* **Inline code:** Use ` ` backticks (e.g., `git push`)
* **Code Block:** Use three backticks ``` before and after the code.

---

## 5. Other Tools
* **Link:** `[Title](URL)`
* **Quote:** `> This is a quote`
